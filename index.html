⚙️ Firebase no configurado
Para activar la sincronización, edita el archivo firebase-sync.js y reemplaza FIREBASE_CONFIG con las credenciales de tu proyecto Firebase.
Pasos rápidos:
1. Ve a console.firebase.google.com
2. Crea un proyecto → Agrega app Web
3. Copia el firebaseConfig en firebase-sync.js
4. Activa Authentication (Email + Anónimo)
5. Activa Firestore Database
💡 Mientras tanto, tus datos se guardan con 💾 Guardar de forma local (localStorage + archivo HTML).
Inicia sesión para sincronizar tus datos entre tu laptop, celular y cualquier dispositivo automáticamente.
